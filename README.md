## Programming Language

Custom simplified programming language model.

#### Variables
```
datum var1;
datum var2: 3;
datum var3: true;
datum var4: "string value";
```

#### Simple math
```
datum celsius: 36.6;
datum fahrenheit: (celsius * 1.8) + 32;
write(fahrenheit);
```

#### Typed core
```
write(Type);
write(Function);
write(Number);
write(Boolean);
write(String);
```
